# Build GO Lang

```go

go build

go build -o novoNome
```

Variaveis de ambiente:

- *GOOS* - Define a plataforma do build, sendo as plataforma disponíveis ['windows', 'linux']
- *GOARCH* - Define a arquitetura sendo ela 32bit ou 64bit, sendo disponíveis ['x36', 'x64', 'arm']
